This lab is the workshop's core exercise. You will run an agentic loop on a real seeded bug and log the result.

## Why this matters

First contact with loops must succeed often enough to build trust. This task is tuned for 3–5 iterations with a clear verifier — not a toy kata, not a production fire drill.

## What you'll learn

- You can baseline iteration 0 (verifier red)
- You can run the act→verify cycle without becoming the feedback loop
- You can recognize blocked vs green termination
- You can write a complete FR-4 metric log

## Common mistakes

### Symptom-only fix
**Symptom:** Primary test green; suite fails.  
**Fix:** Regression gate is part of definition of done — loop continues.

### Human becomes verifier
**Symptom:** You run tests but agent does not.  
**Fix:** Re-prompt with explicit "you must run verifier after each edit."

### Thread pollution
**Symptom:** Earlier demo context skews results.  
**Fix:** New Agent thread for lab.
